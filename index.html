HTML&CSS

HTMLは文章の構造や見た目を表現する言語で、CSSは文章の見た目を整える言語です。
HTMLとCSSだけでプログラムをつくることはできませんが、
様々なプログラミング言語と組み合わせることで、Webを形作る重要な言語です。

JavaScript

スクリプト言語と言い、プログラミングが初めてという方にも
短期間で習得可能な、動的なウェブサイト構築に用いるプログラミング言語です。

jQuery

JavaScriptを簡単に扱えるように、よく使用する機能をまとめた
JavaScriptライブラリというものの１つで、
これを用いることで、効率の良い制作を行うことができます。
jQueryでは、効果やアニメーションを実装可能に！

PHP

静的なWebページの表現しかできないHTMLを、
動的にするために使われるツール群として生まれた言語。
文法が比較的平易で敷居が低い、歴史が長いため、多くのWebサイトで使用されています。

Ruby

オブジェクト指向型のスクリプト言語で、
短く直感的にかけることから、人気の高い言語。
プログラマがストレス無く、プログラミングを楽しめるように設計されている。

Rails

Rails（Ruby on Rails）は、rubyを使ってWebアプリケーションを
より簡単に開発できる フレームワーク。
数多くのWebサイトやWebアプリケーションがRuby on Railsによって開発されている。
